# Dataset Overview

The local corpus is intentionally small and synthetic so the full experiment can be run end-to-end without external storage or hosted infrastructure.

## Corpus Domains
- company policies
- fake product manuals
- academic-style article fragments

## Benchmark
- 60 questions total
- 40 answerable
- 20 unanswerable

The benchmark is generated by [`make_dataset.py`](/c:/Users/ANISH%20PC/Desktop/RAG/rag-hallucination-study/make_dataset.py).

## Why Synthetic Data?
This repository is designed as a reproducible evaluation system. A synthetic corpus keeps:

- licensing simple
- behavior inspectable
- failure cases easy to reason about
- local execution fast

The point of the repository is not broad factual coverage. The point is to make grounded-vs-ungrounded answer behavior measurable.
